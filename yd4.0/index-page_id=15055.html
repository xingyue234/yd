公众号迁移对公众号的要求与影响


原账号条件


个人主体无要求；其他主体必须是小额验证打款过的或者移动互联网认证过的主体


目标账号条件


目标账号必须是小额验证打款过的或者移动互联网认证过的主体，不可以是个人主体


单向迁移


“个人类公众号”可以迁移到“组织类公众号”，但是不支持“组织类公众号”迁移到“个人类公众号”
